# THRONE · VIP Gaming Club

Premium dark-royal casino dashboard prototype — Vue 3 · Vite · TypeScript · Tailwind CSS · GSAP.
Frontend only: no backend, no real payments, no gambling logic.

```bash
npm install
npm run dev    # http://localhost:5173
npm run build
```
